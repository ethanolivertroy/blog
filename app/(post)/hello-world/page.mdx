## Features

This blog template includes:

- **MDX Support** - Write in markdown with React components
- **Tailwind CSS** - Utility-first styling
- **Geist Font** - Clean typography from Vercel
- **Dark Mode** - Automatic theme switching
- **TypeScript** - Full type safety

## Adding Posts

1. Create a new folder in `app/(post)/your-post-slug/`
2. Add a `page.mdx` file with your content
3. Update `app/posts.json` with the post metadata
